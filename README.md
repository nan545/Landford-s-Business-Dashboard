 Business Dashboard Web App 

A responsive **Business Dashboard Web Application** built using **HTML, CSS, and Vanilla JavaScript**.  
The app includes **mock authentication**, **client management (CRUD)**, **analytics with Chart.js**, **search & filter**, and **localStorage persistence** ‚Äî all without any frameworks.

---

 Features

 Authentication
- Mock login system
- Session handled using `localStorage`
- Redirect protection for unauthenticated users

 Client Management (CRUD)
- Add new clients
- Edit existing clients
- Delete clients
- View all clients in a table

 Search & Filter
- Search clients by name in real time

 Analytics (Chart.js)
- Bar chart: Active vs Inactive clients
- Pie chart: Client distribution
- Charts update automatically on data change

 Dashboard Statistics
- Total number of clients
- Number of active clients

 Data Persistence
- All data stored in browser `localStorage`
- Data remains after refresh or browser restart

 UI & UX
- Clean and minimal dashboard design
- Fully responsive layout
- Modal-based form for client management
